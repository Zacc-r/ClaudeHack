# OpenClaw Runs

This directory contains OpenClaw run artifacts, organized by date.

## Structure

```
openclaw-runs/
├── 2026-02-21/
│   ├── run-143022.json
│   └── run-153045.json
└── README.md
```

## Output Contract

- All OpenClaw run logs go here
- Use date-stamped subdirectories: `YYYY-MM-DD/`
- Never create files at repo root
- This directory is gitignored (ephemeral artifacts)
